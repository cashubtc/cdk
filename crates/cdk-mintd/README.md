
# cdk-mintd

## Building from source

```sh
  nix develop
```

#### Build

```sh
  cargo b --bin cdk-mintd -r
```

## Configuration

### Copy example config to cdk-mintd working directory.

```sh
  cp ./example.config.toml ~/.cdk-mintd/config.toml
```

### Edit config file

```sh
  vi ~/.cdk-mintd/config.toml
```
